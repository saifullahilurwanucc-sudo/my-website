Darasi na 10: HTML Summary

Takaitaccen Bayanin HTML

👨‍🏫 Malami:

A yau za mu duba duk abubuwan da muka koya a darussan HTML.

👨‍🎓 Ɗalibi:

Malam, wane abubuwa muka koya?

👨‍🏫 Malami:

Mun koyi yadda ake amfani da tags wajen gina website.

Muhimman HTML Tags

Tag Aiki
<html> Tushen HTML document.
<head> Bayanan shafi.
<title> Sunan shafi.
<body> Abubuwan da ake gani.
<h1> - <h6> Headings.
<p> Paragraph.
<a> Link.
<img> Saka hoto.
<ul> da <ol> Lists.
<table> Tebura.
<form> Karɓar bayanai.

HTML Attributes

Attributes suna bada ƙarin bayani ga tags.

Attribute Aiki
href Adireshin link.
src Wurin hoto.
alt Bayanin hoto.
width Faɗin hoto.
height Tsayin hoto.
border Iyakokin table.

Misalin Cikakken HTML Page

<!DOCTYPE html>
<html>
<head>
<title> My Website </title>
</head>
<body>
<h1> Sannu Duniya </h1>
<p> Wannan shine shafina. </p>
</body>
</html>
