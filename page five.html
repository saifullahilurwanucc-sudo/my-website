Darasi na 5: HTML Lists

Jerin Abubuwa a HTML

👨‍🏫 Malami:

A yau za mu koyi yadda ake tsara abubuwa cikin jerin tsari a HTML.

👨‍🎓 Ɗalibi:

Malam, mene ne List?

👨‍🏫 Malami:

List yana nufin jerin abubuwa da aka tsara ta hanya mai kyau. Misali: jerin sunaye, kayan aiki, ko matakai.

1. Unordered List (<ul>)

Unordered List yana amfani da alamar bullet (•) wajen nuna abubuwa.

Misali:

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

Sakamako:

  • HTML
  • CSS
  • JavaScript

2. Ordered List (<ol>)

Ordered List yana amfani da lambobi wajen tsara abubuwa.

Misali:

<ol>
  <li>Fara koyon HTML</li>
  <li>Koyi CSS</li>
  <li>Koyi JavaScript</li>
</ol>

Sakamako:

  1. Fara koyon HTML
  2. Koyi CSS
  3. Koyi JavaScript

👨‍🎓 Ɗalibi:

Malam, menene aikin <li>?

👨‍🏫 Malami:

<li> yana nufin List Item. Shi yake nuna kowane abu da yake cikin list.

3. Nested List (List a cikin List)

Nested List yana nufin sanya wani list a cikin wani list.

Misali:

<ul>
  <li> Programming
    <ul>
      <li>Java</li>
      <li>Python</li>
    </ul>
  </li>
</ul>

Sakamako:

  • Programming
    • Java
    • Python

4. Definition List (<dl>)

Definition List ana amfani da shi wajen bayyana ma'anar kalmomi.

Yana amfani da:

  • <dl> - Fara Definition List
  • <dt> - Sunan kalma
  • <dd> - Bayanin kalma

Misali:

<dl>
  <dt>HTML</dt>
  <dd> Harshen gina website </dd>
</dl>

Sakamako:

HTML
Harshen gina website.
CSS
Harshen ƙawata website.
JavaScript
Harshen bada aiki ga website.

Bambancin Lists

Tag Aiki
<ul> List mai bullet.
<ol> List mai lambobi.
<li> Abu guda a cikin list.
<dl> Definition List.
<dt> Sunan kalma.
<dd> Bayanin kalma.

Takaitawa

  • <ul> yana yin list mai bullet.
  • <ol> yana yin list mai lambobi.
  • <li> yana wakiltar abu guda.
  • Nested list yana sanya list cikin list.
  • <dl>, <dt>, <dd> suna bayyana ma'anar kalmomi.

Aikin Gida

  1. Ƙirƙiri unordered list na kayan makaranta.
  2. Ƙirƙiri ordered list na matakan koyon HTML.
  3. Yi nested list na programming languages.
  4. Yi definition list na HTML, CSS da JavaScript.
