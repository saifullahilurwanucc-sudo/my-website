Darasi na 4: Links da Images

Haɗa Shafuka da Saka Hotuna a HTML

👨‍🏫 Malami:

A yau za mu koyi yadda ake saka links da hotuna a cikin HTML page.

👨‍🎓 Ɗalibi:

Malam, mene ne Link?

👨‍🏫 Malami:

Link hanya ce da take kai mutum zuwa wani shafi ko website. Ana amfani da tag ɗin: <a>

Anchor Tag (<a>)

Misalin Link:

<a href="https://www.google.com"> Google </a>

Sakamako:

Google

Bayanin href

href yana nufin inda link zai kai. Misali:

YouTube

Buɗe Link a Sabon Shafi

Ana amfani da: target="_blank"

Misali:

Wikipedia

👨‍🎓 Ɗalibi:

Malam, yaya ake saka hoto a HTML?

👨‍🏫 Malami:

Ana amfani da tag ɗin: <img>

Image Tag (<img>)

Misali:

<img src="hoto.jpg" alt="Hoto">

<img> baya buƙatar rufe tag kamar sauran tags.

Misalin Saka Hoto

A nan:

  • src yana nuna inda hoton yake.
  • alt yana bada bayanin hoto.
  • width yana bada faɗin hoto.
  • height yana bada tsayin hoto.

Haɗa Hoto da Link

Ana iya sanya hoto ya zama link.

Misali:

<a href="website.html">
  <img src="hoto.jpg">
</a>

Misalin Cikakken Code

<html>
<body>
<h1> Shafin Nawa </h1>

<a href="about.html"> Game da Mu </a>

<img src="photo.jpg" alt="Hoto">
</body>
</html>

Bambancin Tags

Tag/Attribute Aiki
<a> Ƙirƙirar Link
href Adireshin Link
target="_blank" Buɗe sabon tab
<img> Saka Hoto
src Inda hoto yake
alt Bayanin hoto
width Faɗin hoto
height Tsayin hoto

Takaitawa

  • <a> yana ƙirƙirar link.
  • href yana nuna inda link zai kai.
  • target="_blank" yana buɗe sabon shafi.
  • <img> yana saka hotuna.
  • src yana nuna wurin hoton.
  • alt yana bada bayanin hoto.
  • width da height suna canza girman hoto.

Aikin Gida

  1. Ƙirƙiri link zuwa Google.
  2. Ƙirƙiri link zuwa wani shafi naka.
  3. Saka hoto guda a HTML.
  4. Amfani da src da alt a hoton.
