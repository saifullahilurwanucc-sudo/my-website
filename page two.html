Darasi na 2: HTML Document Structure

Tsarin HTML Page

👨‍🏫 Malami:

A darasin farko mun koyi menene HTML. Yanzu za mu koyi tsarin HTML document.

👨‍🎓 Ɗalibi:

Malam, shin kowane HTML page yana da tsari ne?

👨‍🏫 Malami:

Eh. Dukkan HTML page yana da babban tsari guda. Yana farawa daga:

  • DOCTYPE
  • html
  • head
  • body

Misalin HTML Structure

Ga misalin tsarin HTML:

<!DOCTYPE html>
<html>
  <head>
    <title> My Page </title>
  </head>

  <body>
    Hello World
  </body>
</html>

Bayanin Kowanne Sashe

1. DOCTYPE

DOCTYPE yana gaya wa browser cewa HTML5 ake amfani da shi.

2. HTML Tag

<html> shine tushen document. Duk wani HTML code yana cikin wannan tag.

3. Head Tag

<head> yana dauke da bayanan shafi. Misali: title da meta.

4. Body Tag

<body> yana dauke da abubuwan da mai amfani zai gani. Misali: rubutu, hotuna, tables da links.

Tambayar Ɗalibi

👨‍🎓 Ɗalibi:

Malam, me zai faru idan ban saka body ba?

👨‍🏫 Malami:

Browser zai iya samun matsala wajen nuna abubuwan shafi. Saboda haka body yana da muhimmanci.

Misalin Cikakken HTML Page

<!DOCTYPE html>
<html>
<head>
<title> Website Na </title>
</head>

<body>
<h1> Sannu Duniya </h1>
<p> Wannan shine website na farko. </p>
</body>
</html>

Takaitawa

  • DOCTYPE yana bayyana nau'in HTML.
  • HTML shine tushen page.
  • Head yana dauke da bayanan shafi.
  • Title yana bada sunan page.
  • Body yana dauke da abubuwan da ake gani.

Aikin Gida

  1. Rubuta tsarin HTML document.
  2. Menene aikin head?
  3. Menene aikin body?
  4. Me yasa ake amfani da DOCTYPE?
