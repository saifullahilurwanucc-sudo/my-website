Darasi na 1: Menene HTML?

Koyon HTML da Hausa

👨‍🏫 Malami:

Assalamu Alaikum dalibai. A yau za mu fara koyon HTML.

👨‍🎓 Ɗalibi:

Wa alaikumus salam Malam. Mene ne HTML?

👨‍🏫 Malami:

HTML yana nufin HyperText Markup Language. Shi ne harshen da ake amfani da shi wajen gina shafukan yanar gizo (Web Pages).

👨‍🎓 Ɗalibi:

Malam, HTML Programming Language ne?

👨‍🏫 Malami:

A'a. HTML ba Programming Language ba ne. Markup Language ne. Aikinsa shine tsara yadda abubuwa za su bayyana a website.

Me ake amfani da HTML?

Ana amfani da HTML wajen:

  • Ƙirƙirar Website.
  • Rubuta bayanai a shafi.
  • Saka hotuna.
  • Ƙirƙirar links.
  • Ƙirƙirar tables.
  • Ƙirƙirar forms.

Misalin HTML Tag

HTML yana amfani da Tags.

<h1>Sannu Duniya</h1>

Sannu Duniya

Wasu Muhimman HTML Tags

Tag Aiki
<html> Fara HTML document.
<head> Bayanan shafi.
<title> Sunan shafi.
<body> Abubuwan da ake gani.
<h1> Babban heading.
<p> Paragraph.

Takaitawa

  • HTML yana nufin HyperText Markup Language.
  • Ana amfani da shi wajen gina websites.
  • HTML yana amfani da tags.
  • HTML ba Programming Language ba ne.

Aikin Gida

  1. Rubuta cikakken ma'anar HTML.
  2. Shin HTML Programming Language ne?
  3. Rubuta HTML tags guda biyar.
